# Face Unlock System 🔐

A real-time face unlock authentication system built using Python, OpenCV, and face recognition techniques.

## 🚀 Features
- Face registration
- Real-time face authentication
- Secure access control

## 🛠️ Tech Stack
- Python
- OpenCV
- NumPy

## 📁 Project Structure
- src/ → source code
- data/ → datasets
- models/ → saved models
